# Mini Healthcare Support Web App

## Overview
This project is a concept-level healthcare support application for NGOs and healthcare volunteers.

## Features
- Patient Support Form
- Auto-generated Issue Summary
- Responsive User Interface

## AI / Automation Idea
The application automatically generates a concise summary of the patient's issue after form submission. This helps volunteers quickly understand and manage support requests.

## Tech Stack
- HTML
- CSS
- JavaScript

## NGO Use Case
NGOs can use this application to collect patient concerns and automatically summarize requests for faster response and better record management.
